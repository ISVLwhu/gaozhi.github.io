Computer Vision (计算机视觉)

Quick links: Schedule, Materials, Policies
Course image / 课程图片

This course introduces fundamental concepts and modern practices in computer vision, including image formation, feature extraction, deep learning-based recognition, detection, segmentation, and multi-view geometry.

本课程系统介绍计算机视觉基础与前沿方法，涵盖成像模型、特征提取、基于深度学习的识别/检测/分割以及多视几何等内容。

  • Key topics: CNN/Transformer vision models, detection & segmentation, SfM/MVS basics
  • 核心内容：视觉深度模型、目标检测与分割、SfM/MVS 基础
  • Assessments: homework + project + presentation
  • 考核方式：作业 + 课程项目 + 汇报展示
